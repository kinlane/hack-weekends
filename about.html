About Hack Weekends

This site is intended to provide all the necessary resources for organizing and publizing hackathons. This entire site is hosted on Github as an open source repository. I needed a way to share the information available on Hack Weekends, while sharing the responsibility of maintaining with the community.

To deploy this site I used:

  • Github Pages
  • Jekyll
  • JSON Data Stores
  • Mustache Templates

I wanted an easy way to host pages and the blog for Hack Weekends. Github Pages + Jekyll does this, allowing me to edit these static pages using prose.io.

In addition to pages and blog posts I wanted to share dynamic data like the hackathon events calendar, organizers, venues and much more. JSON files + Mustache templates will handle this.

Here are the hackathon data resources I intend to publish on Hack Weekends:

  • Hackathons
  • Organizers
  • Sponsors
  • Venues
  • Vendors
  • Tools
  • Speakers
  • PR Opportunities

These hackathon resources are available via JSON files, and while browsing via the Hack Weekends site. The goal is to provide open marketplace for hackathons to occur, while running it as an open source project.
